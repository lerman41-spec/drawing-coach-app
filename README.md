# יומן הרישום שלי - גרסה עצמאית

זו גרסה מלאה של האפליקציה, כולל עריכת תמונה אמיתית עם OpenAI. בניגוד לגרסת ה-artifact
בתוך Claude, האתר הזה **לא רץ בחינם** - הוא משתמש בשני מפתחות API בתשלום, שלך בלבד:

- **ANTHROPIC_API_KEY** - לניתוח הרישום ולמשוב (Claude)
- **OPENAI_API_KEY** - לעריכת התמונה האמיתית (GPT Image 2)

שני המפתחות יושבים רק בצד השרת (בתיקיית `pages/api/`) ולעולם לא נחשפים לדפדפן.

אין צורך לכתוב או להבין קוד כדי לפרוס את זה - רק לעקוב אחרי השלבים למטה.

---

## שלב 1 - מפתח Anthropic

1. היכנסי ל-console.anthropic.com והתחברי עם החשבון שלך.
2. בתפריט הצד לחצי על **API Keys**.
3. לחצי **Create Key**, תני לו שם (למשל `drawing-coach`), ולחצי צור.
4. **העתיקי את המפתח מיד** - הוא מוצג פעם אחת בלבד. שמרי אותו זמנית בקובץ טקסט פרטי.
5. ודאי שיש אמצעי תשלום מוגדר תחת **Billing** (זו קריאת API בתשלום, לא חלק מ-Claude.ai).
6. אם השגיאה "model not found" מופיעה מאוחר יותר, בדקי את שמות המודלים העדכניים במסמכי
   docs.claude.com - שמות מודלים משתנים מדי פעם, וייתכן שצריך לעדכן את המחרוזת
   `MODEL` בקובץ `pages/api/analyze.js`.

## שלב 2 - מפתח OpenAI

1. היכנסי ל-platform.openai.com והתחברי.
2. תחת **Settings -> Organization -> General**, אם מופיע כפתור **Verify Organization** -
   לחצי עליו והשלימי את אימות הזהות (תעודה מזהה). זה נדרש כדי להשתמש במודלי GPT Image.
   האימות יכול לקחת עד כ-15 דקות עד שהוא נכנס לתוקף.
3. תחת **API keys**, לחצי **Create new secret key**, תני שם, העתיקי ושמרי אותו זמנית.
4. ודאי שיש אמצעי תשלום תחת **Billing**.

## שלב 3 - העלאת הקוד ל-GitHub (בלי טרמינל)

1. היכנסי ל-github.com וצרי חשבון חינמי אם אין לך.
2. לחצי על **+** בפינה -> **New repository**. תני שם כמו `drawing-coach-app`, השאירי
   אותו Private, ולחצי **Create repository**.
3. בדף הריפו הריק, לחצי על הקישור **uploading an existing file**.
4. גררי לתוך הדפדפן את **כל** הקבצים והתיקיות שקיבלת ממני (`pages/`, `package.json`,
   `next.config.js`, `.gitignore`, `.env.local.example`, `README.md`) - שמרי על מבנה
   התיקיות כפי שהוא.
5. לחצי **Commit changes** בתחתית העמוד.

## שלב 4 - חיבור ל-Vercel ופריסה

1. היכנסי ל-vercel.com וצרי חשבון חינמי - הכי קל להתחבר עם אותו חשבון GitHub.
2. לחצי **Add New -> Project**.
3. תבחרי את הריפו `drawing-coach-app` שיצרת ולחצי **Import**.
4. לפני שלוחצים Deploy, תפתחי את **Environment Variables** ותוסיפי שני שורות:
   - Name: `ANTHROPIC_API_KEY`   Value: (המפתח משלב 1)
   - Name: `OPENAI_API_KEY`      Value: (המפתח משלב 2)
5. לחצי **Deploy** וחכי כדקה-שתיים.
6. בסיום תקבלי קישור אמיתי (`your-app.vercel.app`) - זה האתר החי שלך.

## עדכון בעתיד

כל שינוי שתרצי (גם אם אני אכתוב אותו) - מעלים את הקובץ המעודכן לאותו ריפו ב-GitHub
(**Add file -> Upload files**, לבחור להחליף את הקובץ הקיים), ו-Vercel פורס אוטומטית
מחדש תוך דקה.

## עלויות בערך

- ניתוח רישום אחד (Claude): שברירי סנט עד כמה סנטים, תלוי באורך המשוב.
- עריכת אזור אחת (GPT Image 2, איכות low/medium): בערך $0.005-$0.05 לעריכה.
- אין עלות קבועה חודשית - משלמים רק על שימוש בפועל, וניתן להגדיר תקציב/הגבלה
  בדשבורד של כל ספק.

## פתרון תקלות נפוצות

- **"Your organization must be verified"** (OpenAI) - חזרי לשלב 2.2, ולפעמים צריך
  לחכות כמה דקות אחרי האימות.
- **"model not found"** (Anthropic) - עדכני את שם המודל ב-`pages/api/analyze.js`
  לפי docs.claude.com.
- **שגיאה כללית באפליקציה** - לחצי על כפתור ה-**Deployments** ב-Vercel ותפתחי את
  ה-deployment האחרון כדי לראות את הלוגים - אפשר להדביק לי את הטקסט ואעזור לפענח.
